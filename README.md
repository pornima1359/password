## Description
This project is a **Password Strength Checker** written in Python.  
It checks whether a password is **weak, medium, or strong** based on different rules using regular expressions.

## Features
- Checks minimum password length
- Verifies use of:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Uses Python `re` (regular expressions)
- Beginner-friendly logic

## How It Works
The program analyzes the password and determines its strength by matching patterns using regular expressions.
